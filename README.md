# 📘 Lógica de Programação com PHP

Este repositório foi criado com o objetivo de armazenar conteúdos, exercícios e exemplos desenvolvidos durante as aulas de **Lógica de Programação utilizando PHP**.

## 🚀 Objetivo

Organizar e acompanhar a evolução do aprendizado em lógica de programação, aplicando os conceitos fundamentais com a linguagem PHP.

## 📂 Conteúdo

Neste repositório você encontrará:

- 📌 Conceitos básicos de lógica de programação
- 🔢 Variáveis e tipos de dados
- 🔁 Estruturas de repetição (for, while, do while)
- 🔀 Estruturas condicionais (if, else, switch)
- 📥 Entrada e saída de dados
- 🧮 Operadores matemáticos e lógicos
- 📚 Exercícios práticos
- 💡 Desafios propostos em aula

## 🛠️ Tecnologias Utilizadas

- PHP
- Editor de código (VS Code, por exemplo)
- Git e GitHub
